## How to run this application
Open a `terminal`
```
git clone https://github.com/briighter/palace.git
cd palace
```
Open two individual `terminals`
```
// Terminal 1
cd palace/palace
ionic serve
```

```
// Terminal 2
cd palace/palace-api
npm run start
```
##### Navigate to localhost:3000
[Local Host](http://localhost:3000/)
